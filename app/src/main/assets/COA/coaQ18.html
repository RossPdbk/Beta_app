Explian flynn's classification.

Flynn's classification for computer architectures is based on the number of instruction steams and data streams also called as Flynn's Taxonomy.
An instruction stream is a sequence of instruction while a data stream is a sequence of data which includes input data or temporary results.
They are classified into 4 types:

  • SISD (Single Instruction Single Data)
  • SIMD (Single Instruction Multiple Data)
  • MISD (Multiple Instruction Multiple Data)
  • MIMD (Multiple Instruction Multiple Data)

SISD (Single Instruction Single Data)
  • A single-processor computer (Conventional computer) in which a single stream of instructions is fetched and executed using single data stream.
  • It has no parallelism either in instructions or in data.

SIMD (Single Instruction Multiple Data)
  • A single stream of instructions is fetched and executed using multiple data stream.
  • So each processing unit can operate independently on a different data stream.
  • This group is dedicated to array processing machines.

MISD (Multiple Instruction Multiple Data)
  • In case of MISD computers, multiple processing units operate on one single-data stream .
  • Few parallel computers fits into this class but in practice, this kind of organization has never been used.

MIMD (Multiple Instruction Multiple Data)
  • Each processor has a separate program.
  • An instruction stream is generated from each program.
  • Each instruction operates on different data.
  • Most parallel computers are based on MIMD architecture. There are two major categories
    • 1)Shared memory multiprocessor.
    2)Message passing multi-computers.
