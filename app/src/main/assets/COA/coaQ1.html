Differentiate between Computer Architecture and Computer Organization .

No. Computer Architecture Computer Organization
1 Computer Architecture is about attributes visible to programmer. Computer Organization is about the operational units and their interconnections.
2 It is interface between hardware and software. It deals with components of connection in a system.
3 It describes what computer does. It describes how computer does a task.
4 while designing computer system architecture is considered first. organization is done on the basis of architecture.
5 It deals with high level design issues. It deals with low level design issues.
6 If there is multiply instruction then it is an architectural issue. Implementation of multiply instruction is an organizational issue.
