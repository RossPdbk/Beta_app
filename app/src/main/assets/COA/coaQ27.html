Differentiate between Hardwired and Micro-programmed control.

Attribute Hardwired Control Micro-programmed Control
Basic The control unit whose control signals are generated by the Hardware through a sequence of instruction is called as Hardwired control unit. The control unit whose control signals are generated by the data stored in Control memory is called Micro-programmed Control unit.
Speed Fast slow
Cost Higher Cheaper
Flexibility Not Flexible, as wiring changes are made in Hardwired Control if there are any changes required. Flexible, as changes are done by updating micro-program in control memory.
Design Complicated Systematic
Design Approach Sequential circuit Programming
Decoding & Sequencing logic Complex Easy
Instruction set size & Handling small size, difficult to handle. large size, easier to handle.
Control Memory Absent Present
Applications RISC microprocessor CISC microprocessor
