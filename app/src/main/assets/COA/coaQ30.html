Explain Amdahl's law.

  • Amdahl's law is a formula which gives the theoretical speedup by running part of it in parallel.
  • Speedup:
    • The speed of a program is the time it takes the program to excecute.
    • This could be measured in any increment of time.
    • Speedup is defined as the time it takes a program to execute in serial (with one processor) divided by the time it takes to execute in parallel (with many processors).
  • A program (or algorithm) which can be parallelized can be split up into two parts:
    1. A part which cannot be parallelized
    2. A part which can be parallelized
