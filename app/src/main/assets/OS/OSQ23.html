Q23. Draw and explain 5 state process model.


  1. New : The process is being created.

  2. Ready : The process is ready to be assigned to processor. All those processes are loaded in RAM & waiting for CPU in ready state.

  3. Running : Instruction are being executed in CPU.

  4. Waiting/Blocked : The process is waiting for seome event to occur. Here all those processes are moved from CPU to waiting state. When CPU becomes free, The process from blocked state move to ready state and then in running state.

  5. Exit : The process has finished execution. It is terminated from CPU & RAM.
