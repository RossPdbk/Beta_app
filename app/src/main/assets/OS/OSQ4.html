Q4. Compare the Monolithic and Microkernels.

No. Microkernels Monolithic Kernels
1. In Microkernels, user service and kernel service are kept in seperate address space In Monolithic kernel, Both User services and Kernel services are kept in same address space.
2. Microkernels are smaller in size. Monolithic kernels are larger in size.
3. Slow execution Fast Execution
4. The microkernel is easily extendible The monolithic kernel is hard to extend
5. If a service crashes, It does affect working of microkernel. If a service crashes, The whole system crashes in monolithic kernel.
6. To write microkernel, More code is required. To write monolithic kernel, less code is required.
7. E.g. QNX, Symbian, L4Linux, k42 etc. E.g. Linux, BSDs, Microsoft windows,Solaris etc.
