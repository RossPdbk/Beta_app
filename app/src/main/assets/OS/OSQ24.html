Q24. Write difference between Paging and Segmentation.


No Paging Segmentation
1. A page is of fixed block size. A segment is of variable block size.
2. Paging may lead to internal fragmentation. Segmentation may lead to external fragmentaion.
3. Page is a physical unit. Segment is a logical unit.
4. The user specified address is provided by CPU & divided into page number & offset. The user specifies each address by two quantities 1. Frame number 2. Offset
5. Hardware decides page size. Segment size is specified by user.
6. Paging involves page table which contains base address of each page. Segmentation involves segment table which contains segment number & offset.
7. Procedure & data cannot be seperated Procedure & data can be seperated
8. Data cannot be shared Data can be shared
9. Data cannot be protected Data can be protected
