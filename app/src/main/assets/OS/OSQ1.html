What is Operating system? Explain different function of OS?

Operating System
  • An operating system is system software which manages, operates and communicates with the computer hardware and software. To complete the execution user program need many resources. The main job of the operating system is to provide resources and services to the user program or user prograam so without a computer operating system, a computer would be useless.

  • An OS acts as an interfare between the user and hardware of the computer and also controls the execution of application programs.

  • Operating system is also called as resource manager
Function of Operating System
  1. Process management
    The process management activities involves control access to shared resources like file, memory etc., Control executionof user application, Scheduling a process, Synchronization, deadlock handling etc.

  2. Memory management
    The memory management activities handled by operating system are : Allocation of memory to the process, Free the memory from process after completion, Reallocation of memory to programs.

  3. File management
    The file management activtities of OS consists of : Creation and deletion of files and directories, Provide access to files and allocation of storage , Maintain back-up files, file security.

  4. Device management
    The device management tasks includes : To open,close and write device drivers , Communicate, control and monitor device driver.

  5. Protection and Security
    The resource of system are protected by OS.User authentication, file attributes such as read, write and back-up of data are used by OS togive necessary protection.

  6. Performs basic computer tasks
    Different peripheral devices such as mouse, keyboards are managed by OS.

  7. User interface or command interpreter
    OS offers an interface between the computer user and computer hardware. The user interface is set of commands or a graphical User interface through which user interacts with the applications and the hardware.
