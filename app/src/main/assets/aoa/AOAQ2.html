Q2. Explain three cases of master method.

The Master method is a general method for solving (getting a closed form solution to) recurrence relations that arise frequently in divide and conquer algorithms, which have the following form:

T(n) = aT(n/b) + f(n)

where a ≥ 1, b > 1 are constants, and f(n) is function of non-negative integer n.

There are three cases.

(a) If f(n) = O(n logb a− ε ), for some ε > 0, then T(n) = Θ(n logba ).

(b) If f(n) = Θ(n logb a ), then T(n) = Θ(n logb a log n).

(c) If f(n) = Ω(n logb a+ ε ) for some ε > 0, and af(n/b) ≤ cf(n), for some c < 1 and for all n greater than some value n 0 , Then T(n) = Θ(f(n)).

Example 1 : T(n) = 4T(n/2) + n
For this recurrence, there are a=4, b=2, and f(n)=n. Thus nlogba is n2, and f(n) is O(n2-ε) for ε=1, and Case 1 applies. Thus T(n) is Θ(n2).

Example 2 : (n) = 4T(n/2) + n2
For this recurrence, there are again a=4, b=2, but now f(n)=n2. Again nlogba is n2, and f(n) is thus Θ(n2), so Case 2 applies. Thus T(n) is Θ(n2 log n).

Example 3 : T(n) = 4T(n/2) + n3
For this recurrence, there are again a=4, b=2, but now f(n)=n3. Again nlogba is n2, and f(n) is thus Ω(n2+ε) for ε=1. Moreover, 4(n/2)3 ≤ kn3 for k=1/2, so Case 3 applies. Thus T(n) is Θ(n3).
