  • With adjacency matrix representation, Floyd's algorithm has a worst case complexity of O(n3) where n is the number of vertices.

  • If Dijkstra's algorithm is used for the same purpose, then with an adjacency list representation, the worst case complexity will be O(nelog n).

  • Thus if e is O(n2), then the complexity will be O(n3log n) while if e is O(n), then the complexity is O(n2log n).
