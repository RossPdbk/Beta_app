Q19. What is Hamiltonian cycle? Write an algorithm to find all Hamiltonian cycles.

  • Let G=(V,E) be connected graph with n vertices.

  • A Hamiltonian cycle is a round trip path along n edges of G that visits every vertex once and returns to its starting position.

  • In other words is a Hamiltonian cycle begins at some vertex v1 and the vertices of G are visited in order v1, v2, ... vn+1 then each vertex is distinct except v1 and vn+1 which are equal.

  • The graph G1 of the given figure contains the Hamiltonian cycle 1, 2, 8, 7, 6, 5, 4, 3, 1.

  • The graph G2 contains no Hamiltonian cycle. There is no known easy way to determine whether a given graph contains a Hamiltonian cycle.
  • We begin by representing function NextValue(k), which determinses the possible next vertex of the proposed cycle.

  • Using NextValue we can particularize the recursive backtracking schema to find all Hamiltonian cycles.

  • This algorithm is started by first initializing the adjacency matrisx G[n,n], then setting x[n]=0 and x[1]=1 then executing Hamiltonian(2).

  • This tour is a Hamiltonian cycle. For the simple case of a graph all of those edge cost are identical, Hamiltonian will find a minimum cost tour if a tour exits.

  • If the common edge cost is c, the cost of a tour is cn since there are n edges in a Hamiltonian cycle.
Algorithm NextValue(k)
1.{
2.  repeat
3.  {
4.      x[k]=(x[k+1] mod (k+1));
5.      if (x[k]=0) then return;
6.      if (G[x[k-1], x[k])!=0)
7.      { //Is there an edge?
8.          for j=1 to k-1
9.          {
10.             if(x[j]=x[k])
11.                 break; //check for distinctness
12.             if(j=k)
13.             //if true then vertex is distinct
14.             {
15.                 if((k<n) or(k=n) and G[x[n],x[1]]!=0)
16.                 return;
17.             }
18.         }
19.     }
20. }until (false);
21.}
