Q3. write a note on openGL?

  • Introduction:
OpenGL(Open Graphics Library) is a standard specification defining a cross language
,cross platform API for writing applications that produce 2D and 3D computer graphics.
  OpenGL is a low level,procedural API, requiring the programmer to dictate the
exact steps required to render a scene.
  • OpenGL Operations:
The processing stages in basic OpenGL operation as follows:
  •   Display List:  Rather than having all commands proceed immediately,through the
    pipeline,you can choose to accumulate some of themin a list for processing later.

  • nbspEvaluater: this stage of processing provides an efficient way to approximate curve and surface geometry

  •  Primitive Assembly: OpenGL processes geometric primitive points,line
    segments and polygons all of which are described by vertices.Vertices are
    transformed and lit, and primitives are clipped to the viewport in preparation for rasterizion.

  •  Rasterization: This stage produces a series of frame buffer addresses and associated
    values.Each fragment so priduced is fed into the last stage.

  •  Pre-fragment Operation: These are final operation performed on the data
    before it stored as pixels in the frame buffer.
