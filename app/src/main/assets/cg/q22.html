Q22.Explain purpose of Inside-Outside test with example

This method is also known as counting number method. While filling an object, we often need to identify whether particular point is inside the object or outside it. There are two methods by which we can identify whether particular point is inside an object or outside.

  • Even-Odd Method
  • Nonzero winding number Method
Even-Odd Method
  • In this technique, we will count the edge crossing along the line from any point (x,y) to infinity.
  • If the number of interactions is odd, then the point (x,y) is an interior point; and if the number of interactions is even, then the point (x,y) is an exterior point.
  • The following example depicts this concept
  1. construct a line segment from the point in question to point outside the polygon.
  2. Now, we draw a line segment from point 1 to a outside point and see that the line intersects the polygon at 4 places.
  3. That’s even, hence the point is outside the polygon.
  4. Same goes with point 2.
  5. Now, Point 3 makes an odd intersection of 1, hence it is inside the polygon.
  6. If the intersection point is vertex of polygon then, if the other end points of the two segments which meets at this vertex lie on same side of constructed line then point is taken as having even number of intersections.
  7. If they lie on the opposite side of constructed line then point is counted to have a single intersection.
Nonzero winding number Method
  • This method is also used with the simple polygons to test the given point is interior or not.
  • It can be simply understood with the help of a pin and a rubber band. Fix up the pin on one of the edge of the polygon and tie-up the rubber band in it and then stretch the rubber band along the edges of the polygon.
  • When all the edges of the polygon are covered by the rubber band, check out the pin which has been fixed up at the point to be test. If we find at least one wind at the point consider it within the polygon, else we can say that the point is not inside the polygon.
  • Alternative way to understand this method is as follows
  1. Draw a scan line from the point to be test towards the left most of X direction.
  2. Give the value 1 to all the edges which are going to upward direction and all other -1 as direction values.
  3. Check the edge direction values from which the scan line is passing and sum up them.
  4. If the total sum of this direction value is non-zero, then this point to be tested is an interior point, otherwise it is an exterior point.
  5. In the above figure, we sum up the direction values from which the scan line is passing then the total is 1 – 1 + 1 = 1; which is non-zero. So the point is said to be an interior point.
